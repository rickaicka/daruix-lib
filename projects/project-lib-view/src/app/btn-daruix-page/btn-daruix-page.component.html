How to use

The tag: <lib-btn-daruix> </lib-btn-daruix>

The @Input() list:
[labelBtn] -> set the label of the button
[typeBtn] -> set the type of the button and set his colors
[icon] -> set the icon of the button*is optional
Example: <lib-btn-daruix [labelBtn]='Red' [typeBtn]='red' [icon]='home'> </lib-btn-daruix>
