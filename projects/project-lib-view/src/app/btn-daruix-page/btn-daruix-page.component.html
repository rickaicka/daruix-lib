How to use

The tag: <drx-button> </drx-button>

The @Input() list:
[labelBtn] -> set the label of the button
[typeBtn] -> set the type of the button and set it's colors
[icon] -> set the icon of the button*is optional
Example: <drx-button [labelBtn]='Red' [typeBtn]='red' [icon]='home'> </drx-button>
