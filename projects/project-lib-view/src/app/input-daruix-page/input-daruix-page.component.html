How to use

The tag: <drx-input> </drx-input>

The @Input() list:
[labelInput] -> set the label of the Input
[colorInput] -> set the type of the Input and set it's colors
[disabledInput] -> set if Input is disabled of the button*is optional
Example: <drx-input [labelInput]='Red' [colorInput]='red' [indeterminateInput]=true [disabledInput]=false> </drx-input>
