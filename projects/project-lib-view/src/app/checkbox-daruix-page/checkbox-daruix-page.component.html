How to use

The tag: <lib-checkbox-daruix> </lib-checkbox-daruix>

The @Input() list:
[labelCheckbox] -> set the label of the checkbox
[typeCheckbox] -> set the type of the checkbox and set it's colors
[indeterminateCheckbox] -> set if checkbox is indeterminate the button*is optional
[disabledCheckbox] -> set if checkbox is disabled of the button*is optional
Example: <lib-checkbox-daruix [labelCheckbox]='Red' [typeCheckbox]='red' [indeterminateCheckbox]=true [disabledCheckbox]=true> </lib-checkbox-daruix>
